# StepGuard Monitor

Panel de control para Windows diseñado para supervisar en tiempo real el estado de conexión de los dispositivos **StepGuard**.

### ¿Cómo funciona?
La aplicación detecta automáticamente los dispositivos y rastrea su disponibilidad mediante pulsos de vida (*heartbeats*):
*   **ONLINE (Verde):** El dispositivo está conectado y funcionando.
*   **OFFLINE (Rojo):** El dispositivo no ha reportado señal en los últimos 15 segundos (posible pérdida de batería o WiFi).
